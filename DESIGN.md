# MyPet Design Guidelines

## Brand Identity
- **App Name**: MyPet
- **Tone**: Warm, friendly, modern, trustworthy
- **Feel**: Like a caring pet companion — approachable but professional

## Color Palette
- **Primary**: Warm teal/green (#2D9CDB or similar — clean, fresh, caring)
- **Accent**: Warm amber/orange (#F2994A — energetic, friendly, pet-loving)
- **Background**: Soft off-white (#F9FAFB) with white cards
- **Text**: Dark charcoal (#1F2937) for readability
- **Success/Health**: Soft green (#27AE60)

## Typography
- **Headings**: Rounded, friendly sans-serif (e.g., Nunito or Poppins)
- **Body**: Clean readable sans-serif (Inter or similar)
- **Size scale**: Generous, comfortable — not cramped

## Components
- Rounded cards with soft shadows (border-radius: 12–16px)
- Warm button styles — filled primary buttons with rounded corners
- Soft hover states — no harsh transitions
- Pet-themed iconography where possible (paw prints, hearts, stethoscope)
- Bottom navigation or sidebar for customer app
- Clean data tables and charts for admin dashboard

## Layout
- **Customer side**: Card-based grid for products/services; clean booking flows
- **Admin side**: Sidebar navigation, dashboard with key metrics, data tables
- **AI Chatbot**: Floating chat widget accessible from all pages
- Mobile-first responsive design
